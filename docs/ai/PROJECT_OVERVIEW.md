# Wibestyle / "Я на стиле" — Project Overview

## Цель проекта
Создать многостраничный лендинг для AI-примерочной товаров с маркетплейсов без посещения офлайн-магазинов.

Главный оффер:
"Примерь одежду с маркетплейса на себе — до покупки".

## Продуктовая суть
- Пользователь загружает фото в полный рост (в облегающей одежде).
- Вставляет ссылку на товар с маркетплейса.
- Получает визуальную AI-примерку вещи на себе до покупки.

## Целевая аудитория
- Основная: девушки 18-40, покупающие одежду на маркетплейсах.
- Вторичная: мужчины (пиджаки, обувь, часы, аксессуары, деловой/casual стиль).

## Назначение сайта
- Ярко презентовать идею продукта.
- Собирать лиды на ранний доступ.
- Быть SEO/GEO-готовым многостраничным лендингом.

## Визуальный вектор
- Ориентир на предоставленный референс-эскиз.
- Fashion-tech стиль: максимум визуала, плотный контент, минимальные пустоты.
- Акцент на "изображения + стиль + CTA", без корпоративной сухости.
