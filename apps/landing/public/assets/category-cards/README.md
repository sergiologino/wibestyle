# Category card photos

Поместите фотографии в эту папку, сохранив зарезервированные имена:

- `dress.png` — Платья
- `shoes.png` — Обувь
- `office.png` — Офис
- `evening.png` — Вечер
- `men.png` — Для него

Вместо PNG можно использовать WebP, JPG, JPEG или AVIF с тем же basename. Приоритет форматов: PNG → WebP → JPG → JPEG → AVIF. После добавления или замены файлов перезапустите dev-сервер либо выполните новую production-сборку.
