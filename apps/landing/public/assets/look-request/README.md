# Look request page assets

Replaceable visuals for `/podbor-obraza` ("Подбор образа по запросу").

Use clear fashion photos, not crops from other landing blocks.

- `full-look.png` — full outfit: clothes, shoes, accessories, makeup/hair in one look.
- `accessories.png` — bag, jewelry, watch, glasses, belt, details of the look.
- `shoes.png` — shoes as part of the outfit; vertical or close-up + outfit context.
- `makeup.png` — makeup and hair styling, preferably a clear portrait or half-body crop.

Recommended format: vertical 3:4 or 4:5, important subject centered, no text baked into the image.
